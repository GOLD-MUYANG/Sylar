# Linux 常用命令学习手册 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `.codex/` 中的真实排障命令整理成一份面向 Linux 初学者、可安全练习的通用命令教程。

**Architecture:** 正文只使用一个手册文件，按“基础操作 → 组合命令 → 排障工具 → 开发工具 → 安全练习”的顺序编排。`.codex/*.md` 只作为命令素材源；重复命令合并讲解，Sylar 路径仅作为实战示例。

**Tech Stack:** Markdown、Bash/GNU/Linux 常用工具、ripgrep、Git、CMake、CTest、curl、OpenSSL

## Global Constraints

- 正文输出路径固定为 `文档/Linux常用命令学习手册.md`。
- 目标读者是能打开 Linux 终端，但对命令、参数、管道和重定向不熟悉的初学者。
- 主体素材来自 `.codex/*.md`，只补充建立完整学习链路所需的少量基础命令。
- 不扩展成 Linux 系统管理、发行版安装或 Shell 脚本编程大全。
- 同一命令因路径、文件名或测试目标不同而重复时，合并为一处讲解。
- 默认示例必须只读，或只修改明确创建的临时练习目录。
- `kill`、`chmod`、`mv`、覆盖重定向、`git reset` 等操作必须标注影响范围和安全替代方案。
- 不写入真实 API Key，不提供针对宽泛目录的递归删除示例。
- 不修改 `.codex/` 源记录、`文档/手动理解项目.md` 或当前工作区中的其他用户改动。
